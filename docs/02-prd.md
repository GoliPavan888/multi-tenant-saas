# Product Requirements Document (PRD)

## Product Name
Multi-Tenant Project & Task Management SaaS

## Target Users
- SaaS administrators
- Organization admins
- Team members

## Core Features
- Multi-tenant organization management
- Project creation & tracking
- Task assignment & status updates
- Role-based access control
- Subscription plan enforcement
- Audit logging

## Subscription Plans
- Free: 5 users, 3 projects
- Pro: 25 users, 15 projects
- Enterprise: 100 users, 50 projects

## Non-Goals
- Billing & payments
- Notifications
- File uploads
